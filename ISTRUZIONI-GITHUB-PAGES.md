https://github.com/7jkdqgyb6w-crypto/Ricettario

## Cosa caricare
Carica il contenuto di questa cartella **nella root del repository**.

## Poi in GitHub
- Settings
- Pages
- Source: Deploy from a branch
- Branch: main
- Folder: /(root)

## URL atteso
https://7jkdqgyb6w-crypto.github.io/Ricettario/

## Dominio personalizzato
Quando vorrai collegare il dominio:
- aggiungi un file `CNAME` nella root del repository
- inserisci nel file il dominio, per esempio:
  `ricettario.tuodominio.it`
- aggiorna il DNS del dominio verso GitHub Pages
